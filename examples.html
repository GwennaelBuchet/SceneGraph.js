ANIMATION

Animate a node is very easy. Almost everything ine animatable : position, transparency, scale, ...

cgSceneGraph provides a complete animation framework, with timelines and key frame.
Animations can be chained or launched in the same time ; keys in timeline can be modified at runtime ; animation can be pre-computed
and shared between several nodes ; Events are fired (onAnimationStart, onAnimationEnd) ; ...

The following examples are very simple (the most simple as I could) just to show the simplicity and the power of the code.
So they are not visually sexy... :(
