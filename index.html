HTML5 SCENEGRAPH
FRAMEWORK

The cgSceneGraph is a powerful, cross-channel and very easy-to-use Scene Graph framework written in JavaScript for the <canvas> tag in HTML5.
It's full object oriented to ease the development of your applications and games.

Since the start of the project, it has been designed to be used on professional application as well as for game development.

Feel free to share your extensions or to publish a link to your projects using the cgSceneGraph :)?
