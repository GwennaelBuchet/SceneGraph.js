A very cool game made by Xymostech in few days for the Ludum Dare 25 Compo.

You can play it here (http://www.rampancylabs.com/ludum25/) and sources are downloadable on GitHub (https://github.com/xymostech/Ludum25)

My Quote Five is a nice and very easy-to-use "image maker" made by George Sokianos (aka Walkero).

Using this app you can create images on the fly.
It uses HTML5, CSS3, Canvas, jQuery, Javascript and cgSceneGraph. It already provides a few images for the background and for icons, for you to use, but you still can upload your images as well. You can also add your texts at the images, by selecting different font family, size and color. If you add a text, you can still change it by selecting it and by using the form at the right.

Use it at http://web-projects.gr/myquotefive
