Global properties

Global properties are accessible by any page or class of your application.

cgsgVersion : String
Current version of the framework.
cgsgDisplayRation : CGSGScale
Current display ratio of the rendering.
DO NOT modify it without using CGSGScene.setDisplayRatio method !
Default: CGSGScale(1.0, 1.0)
cgsgResizeHandleThreshold : float
Threshold (or "space around handlers") to detect the handle boxes on a resizable node.
Increase it if you want to select handle boxes around a selected node at a further distance (very useful in case of touch screen).
Default: 3
cgsgCurrentExplorer : a cgsgExplorerParams object
Encapsulate the current browser used by the user.
At the start of the application, the framework will fill this property.
Possible values:
  • cgsgExplorerParams.IE10
  • cgsgExplorerParams.IE9
  • cgsgExplorerParams.SAFARI
  • cgsgExplorerParams.CHROME
  • cgsgExplorerParams.OPERA
  • cgsgExplorerParams.FIREFOX
  • cgsgExplorerParams.KONQUEROR
  • cgsgExplorerParams.UNKNOWN
Default : cgsgExplorerParams.UNKNOWN
cgsgGhostContext : Canvas 2D Context
The global ghost context for fake rendering.
It is used when need to achieve a fake rendering (ie: only in memory, not on the screen) to compute picknode or anything else...
cgsgCurrentFrame : int
The current frame number.
It is initialized to 0 when the application render its first frame and increased by the framework.
Use it in readonly.

 

Global methods

Global methods are accessible by any page or class of your application.

Array.prototype.without(item)
Remove an item from the array.
@param item A primitive or an object
Example: 
                                    var listNodes = [nodeA, nodeB, nodeC];
                                    listNodes.without(nodeB);
                                    //now listNodes contains [nodeA, nodeB]
Array.prototype.clear()
Empty an array.
Example: 
                                    var listNodes = [nodeA, nodeB, nodeC];
                                    listNodes.clear();
                                    //now listNodes contains []
Array.prototype.clone()
Return a clone of the array.
Example: 
                                    var list1 = [nodeA, nodeB, nodeC];
                                    var list2 = list1.clone();
                                    //now list2 contains [nodeA, nodeB, nodeC]
Array.prototype.contains(item)
Checks whether the specified elements exists in the array or not and return a boolean.
@param item A primitive or an object
Example :
                                    var listNodes = [nodeA, nodeB, nodeC];
                                    listNodes.contains(nodeD); //return false
                                    listNodes.contains(nodeC); //return true
Array.prototype.unique()
Remove all duplicates in the array.
Example: 
                                        var listNodes = [nodeA, nodeB, nodeC, nodeB];
                                        var listNodes.unique();
                                        //now list2 contains [nodeA, nodeB, nodeC]
cgsgGetRealViewportDimension()
Return a CGSGDimension as the real viewport dimension.
@return A CGSGDimension object
cgsgGetDisplayedViewportDimension()
Return a CGSGDimension as the viewport dimension scaled with the current display ratio.
@return A CGSGDimension object
cgsgPointIsInRegion(point, targetRegion, threshold)
Return true if the point is inside the region.
@param point The CGSGPosition to check
@param targetRegion The CGSGRegion to check
@param threshold An int. The point is inside the "targetRegion expanded by threshold". Can be negative.
@return A boolean
cgsgRegionIsInRegion(region, targetRegion, threshold)
Return true if the region is totally inside the region.
@param point The CGSGRegion to check
@param targetRegion The CGSGRegion to check
@param threshold An int. The region is inside the "targetRegion expanded by threshold". Can be negative.
@return A boolean
cgsgGetCursorPosition(event, canvas)
Return the mouse or touch position relative to the canvas.
@param event A touch or mouse Event
@param canvas A handler to the Canvas element
@return A CGSGPosition object
cgsgExist(item)
Return true if the item !== null && !== undefined. Return false otherwise.
@param item An object
@return A boolean
cgsgDetectCurrentExplorer()
Detect the current version of the browser and fill the "cgsgCurrentExplorer" property.

You don't have to call it yourself. Only the framework should call it.
Use the "cgsgCurrentExplorer" property instead.
