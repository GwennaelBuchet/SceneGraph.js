Class CGSGScene

Represent the scene of the application.
It ensapsulates the scenegraph itself and several methods to track mouse and touch events, ...

Your main class should extend this one.

Example:
HTML Page
                                        <canvas id="scene" width="640" height="480">
                                            Your browser does not support the canvas element.
                                        </canvas>

                                        <script type="text/javascript">
                                            var canvasScene = document.getElementById("scene");
                                            var game = new CGMain(canvasScene);
                                        </script>
Main JS class
                                        var CGMain = CGSGScene.extend(
                                            {
                                                initialize : function (canvas) {
                                                    //call the parent initializer (ie: constructor)
                                                    this._super(canvas);

                                                    //create a dummy scene with a dummy object
                                                    this.createScene();

                                                    //call this to start animation, frame counter, ...
                                                    this.startPlaying();
                                                },

                                                /**
                                                 * Just create a single node (a square node)
                                                 */
                                                createScene : function () {
                                                    this.squareNode = new CGSGNodeSquare(60, 20, 200, 200); //(x, y, w, h)
                                                    //add some attributes to the square
                                                    this.squareNode.isResizable = true;
                                                    this.squareNode.isDraggable = true;

                                                    //add the square into the scene graph, as root node
                                                    this.sceneGraph.addNode(this.squareNode, null);
                                                }
                                            }
                                        );

Public properties

canvas : Canvas HTML Element
Handler to the Canvas.

Readonly property.

context : Canvas 2D Context
Handler to the 2D context of the canvas.

Readonly property.

sceneGraph : CGSGSceneGraph
Handler to the scene graph itself.

Readonly property.

selectedNodes : Array of CGSGNode
List of the current selected nodes in the scene graph.
Each of these nodes also have their "isSelected" attribute set to true.

Readonly property.

fps : float
Current FPS (frames per second) of the framework.
If you just want to display the current fps in a HTML container, you should use the "showFPS" method of CGSGScene.
Example in the HTML page:
                                        <canvas id="scene" width="80" height="50">
                                            Your browser does not support the canvas element.
                                        </canvas>
                                        <span id="fps"></span> fps.

                                        <script type="text/javascript">
                                            var canvasScene = document.getElementById("scene");
                                            var game = new CGMain(canvasScene);
                                            game.showFPS(document.getElementById("fps"));
                                        </script>

Readonly property.
